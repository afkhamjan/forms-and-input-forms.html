# Semantic HTML & Forms Practice 

This repository demonstrates the use of **Semantic HTML tags** and **HTML Forms**, including a **Payment Form** built using proper structure and accessibility best practices.

---

## Semantic HTML

Semantic HTML provides meaning to the web page structure, improving **readability, accessibility, and SEO**.

### Common Semantic Tags Used

```html
<header>   <!-- Top section (logo, title, nav) --> </header>
<nav>      <!-- Navigation links --> </nav>
<main>     <!-- Main content of page --> </main>
<section>  <!-- Grouped related content --> </section>
<article>  <!-- Independent content block --> </article>
<aside>    <!-- Side content --> </aside>
<footer>   <!-- Bottom section --> </footer>

common form elements:
<form>
  <input type="text">
  <input type="email">
  <input type="password">
  <textarea></textarea>
  <select></select>
  <button type="submit">Submit</button>
</form>
